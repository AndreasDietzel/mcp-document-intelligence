# Python Scripts für lokale Tests

Diese Python-Scripts sind für lokale Tests mit spezifischen Dateipfaden und werden nicht ins Repository committed.

## Verfügbare Scripts:

- **intelligent-rename.py** - PDF-Analyse und intelligente Umbenennung
- **cleanup-old-structure.py** - Bereinigung alter Ordnerstrukturen
- **optimize-folder-structure.py** - Optimierung der Ordnerhierarchie
- **move-loose-files.py** - Kategorisierung loser Dateien
- **show-final-structure.py** - Anzeige der finalen Archivstruktur

## Hinweis:

Die Funktionalität dieser Scripts ist jetzt im MCP Server (TypeScript) integriert und kann über den MCP Server genutzt werden.

Siehe `src/advanced-tools.ts` für die TypeScript-Implementierung.
